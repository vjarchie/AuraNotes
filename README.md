# Aura Notes ✨
**Interactive Fingerboard & Scale Explorer**

Aura Notes is a premium, modern music theory and practice tool designed for violinists (and soon all string players). It features a brilliant, glowing "Aura" aesthetic and a highly responsive dashboard layout to help you visualize scales, arpeggios, and hand positions in real-time.

## 🚀 Key Features

### 1. Dynamic Fingerboard Visualization
- **Vertical Strings**: A high-fidelity representation of the Violin fingerboard (G, D, A, E).
- **Position Highlighting**: Deep support for hand positions (1st–7th). Aura Notes automatically highlights exactly where each finger should be placed.
- **Aesthetic Overlays**: Translucent glass-panel overlays with uniform finger labeling.

### 2. Auto-Play Engine
- **Rhythmic Practice**: Sequence scales or arpeggios across 1, 2, or 3 octaves.
- **Dynamic Shifting**: The app automatically shifts the visual hand position as you climb into higher registers (e.g., automatically moving to 4th position when playing high D6).
- **Arpeggio Sweep**: Toggle between full scales or specialized arpeggio-only practice.

### 3. Integrated Sheet Music (VexFlow)
- **Real-time Notation**: Every note you click or play is instantly rendered on a professional Treble Clef musical staff.
- **Key Signatures**: Automatically displays the correct sharps/flats for your selected Key Center.

### 4. Master Controls
- **Tempo & Volume**: Precision sliders for BPM and Master Audio levels.
- **Visual Feedback**: Pulsating glow highlights keep you centered on the active note.

## 💻 Installation

Aura Notes is built as a **Progressive Web App (PWA)**, meaning you can install it on your computer without a traditional installer.

### To Install via Google Chrome:
1. Open Aura Notes in Chrome.
2. In the address bar, look for the **Install** icon (shaped like a computer screen with a down arrow).
3. Click "Install". 
4. Aura Notes will now appear in your Taskbar or Applications folder as a standalone app!

## 🛠️ Development

Aura Notes is built with **React**, **TypeScript**, and **Vite**.

### Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run in development mode:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```
